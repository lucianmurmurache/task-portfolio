My Work

1. Project Pokemon

JavaScript, JQuery, Bootstrap

A simple Pokemon application, built from scratch using JQuery library and Bootstrap framework. It shows a list of 150 pokemons, the modals include image, name and height. Data loaded via PokeAPI.

See project Pokemon on Github, or see Demo

2. Project Rotten Potatoes

Node.js, Express, MongoDB, Mongoose, NoSQL

RottenPotatoes provides users with access to information related to different movies, directors, movie genres and actors. Users will be able to register, update their information and create a list of their favourite movies.

See project Rotten Potatoes on Github, or see Demo

3. Project MyAwesomeQuizApp

Angular

The project was built using Angular, it allows the user to select one of the available quizzes, answer the questions (from a JSON file). Once the user has answered the quiz questions, the results will be prompted!

See project MyAwesomeQuizApp on Github, or see Demo

4. Project ToDo

Vue.js

An application created using Vue.js, it allows the user to add todo items, edit the added items, tick them as done or remove them from the list.

See project ToDo on Github, or see Demo

5. Project loverovero

JQuery, Bootstrap, PHP

A simple but responsive website built using JQuery library and Bootstrap framework. The user can access information about the vendor, see a list of products and contact the vendor. The contact page passes the data which the user inputs via PHP and is received and safely stored in the server.

See project loverovero
